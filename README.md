E-Library Management System
A secure and flexible digital library platform built for educational institutions and organizations.
This system allows administrators to efficiently manage digital resources while providing controlled access to users based on roles, departments, classes, or custom groups.
✨ Features

Role-Based Access Control (RBAC) – Granular permissions to control who can read, download, or open media files
Flexible Content Access – Make resources public or restrict them to specific classes, arms, departments, faculties, or user groups
Secure User Authentication – Individual login for students, lecturers, staff, and admins
Powerful Admin Dashboard – Full backend control to manage users, roles, resources, and permissions
Multi-format Resource Support – Handles PDF books, journals, videos, audio files, and other digital materials
Highly Customizable – Easily adaptable to suit the needs of any institution or organization

🛠️ Technology Stack

Backend: PHP with Laravel Framework
Database: MySQL
Frontend: Laravel Blade Templates
